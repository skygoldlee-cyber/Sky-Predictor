# ML 기반 인트라데이 전략 최적화 보고서

## 1. 프로젝트 개요

### 1.1 목표
기존 "BULL + NEUTRAL 레짐 완화 + Half Kelly + 인트라데이" 전략의 수익성을 머신러닝 기법을 통해 향상

### 1.2 기간
- 데이터 준비: 2019-2026년 (8년)
- 구현 기간: 1주
- 모델: XGBoost, Random Forest, LSTM, Kelly Criterion

### 1.3 대상 전략
- 기존 승률: 51.08%
- 기존 총 PnL: 996만 원
- 기존 평균 PnL: 6,551 원/건

## 2. ML 최적화 단계

### 2.1 1단계: 데이터 준비

#### 목표
기존 백테스트 데이터 추출 및 피쳐 엔지니어링

#### 수행 내용
- **데이터 추출**: 2019-2026년 KOSPI200 5분봉 데이터
- **기술적 지표**: RSI, MACD, ATR, ADX, SuperTrend, MA20/60, Bollinger Bands
- **시장 데이터**: 진입/청산 시점의 가격, 거래량
- **시간 패턴**: 시간, 요일, 월
- **레짐 정보**: BULL/NEUTRAL 레짐

#### 결과
- 총 거래 수: 1,521건
- 승률: 51.08%
- 총 PnL: 996만 원

#### 주의사항
- 완화된 파라미터 사용 (필터 비활성화, min_wave_pct=0.1, min_pivot_interval_bars=5)
- BULL 최적 파라미터로는 일별 리셋 모드에서 피봇 검출 불가

#### 모델 구성
- **알고리즘**: XGBoost Classifier
- **파라미터**: n_estimators=100, max_depth=6, learning_rate=0.1

- **재현율**: 0.7778

#### 피쳐 중요도 (상위 10)

5. entry_month
6. entry_hour

#### 결론
- 목표 초과 달성 (승률 92.40%)

### 2.4 4단계: 청산 타이밍 최적화 (LSTM)

#### 목표
시계열 패턴 학습을 통해 청산 시점 최적화

#### 모델 구성
- **알고리즘**: LSTM (Long Short-Term Memory)
- **구조**: LSTM(64) → Dropout(0.2) → LSTM(32) → Dropout(0.2) → Dense(16) → Dense(1)
- **시퀀스 길이**: 10
- **학습 Epochs**: 50 (Early Stopping 적용)

#### 성능
- **테스트 정확도**: 0.9531
- **정밀도**: 0.9531
- **재현율**: 1.0000
- **F1 점수**: 0.9760

#### 결과 (Threshold 0.7)
- 거래 수: 650 → 640 (-2%)
- 승률: 95.38% → 95.47% (+0.08%p)
- 총 PnL: 4,509만 원 → 4,480만 원 (-1%)
- 평균 PnL: 69,367 원 → 70,008 원 (+1%)

#### 결론
- 승률 95.47% 달성
- PnL 소폭 감소하지만 평균 PnL 증가

---

### 2.5 5단계: 포지션 사이징 최적화 (Kelly Criterion)

#### 목표
Kelly Criterion을 통한 최적 포지션 사이징

#### Kelly Criterion 계산
- 승률 (p): 0.9547
- 패배 확률 (q): 0.0453
- 평균 승리 (포인트): 2.48
- 평균 패배 (포인트): 3.19
- 승/패 비율 (b): 0.7767
- **Kelly 비율: 0.8964**

#### 다양한 Kelly 비율 테스트

| 전략 | Multiplier | 총 PnL (원) | 평균 PnL (원) |
|------|------------|------------|--------------|
| Full Kelly | 28,235 | 4,016만 원 | 62,751 원 |
| Half Kelly | 14,118 | 2,008만 원 | 31,376 원 |
| Quarter Kelly | 7,059 | 1,004만 원 | 15,688 원 |
| Eighth Kelly | 3,529 | 502만 원 | 7,844 원 |
| Fixed (Current) | 31,500 | 4,480만 원 | 70,008 원 |

#### 결론
- 기존 Fixed multiplier(31,500)가 최적
- 승/패 비율이 1 미만이므로 Kelly Criterion 적용 시 수익 감소
- 기존 포지션 사이징 유지

### 6.2 기술적 발견

#### 피쳐 중요도
- **SuperTrend 방향**: 가장 중요한 피쳐
- **RSI**: 과매수/과매도 상태 중요

- **LSTM**: 시계열 패턴 학습 가능하지만 개선 효과 제한적

### 6.3 제언

#### 1. BULL 최적 파라미터 재검토
- 현재 완화된 파라미터 사용 중
- BULL 최적 파라미터로 피봇 검출 불가 문제 해결 필요
- 일별 리셋 모드 대신 다른 접근 방식 고려

#### 2. Kelly Criterion 재검토
- 현재 승/패 비율이 1 미만으로 Kelly 적용 시 수익 감소
- 손절/익절 비율 개선을 통해 Kelly Criterion 활용 가능성 검토

#### 3. 추가 피쳐 고려
- 시장 마이크로구조 피쳐 (호가 불균형 등)
- 뉴스/이벤트 데이터
- 다른 자산 클래스와의 상관관계

#### 4. 앙상블 모델 고려
- XGBoost, Random Forest, LSTM 앙상블
- 스태킹(Stacking) 또는 블렌딩(Blending)

#### 5. 실시간 적용 테스트
- 페이퍼 트레이딩
- 소규모 실전 테스트
- 모델 드리프트 모니터링

### 6.4 리스크 관리

#### 과적합 리스크
- 학습 데이터(2019-2025)와 테스트 데이터(2026) 성과 차이 확인 필요
