# 승/패 비율 개선 제안서

## 1. 현재 문제 분석

### 1.1 현재 승/패 비율
- **승리 시 평균 수익**: 2.48 포인트
- **패배 시 평균 손실**: 3.19 포인트
- **승/패 비율**: 0.7767 (승리 시 평균 수익 < 패배 시 평균 손실)

### 1.2 문제점
- 승리할 때 적게 이기고, 질 때 크게 짐
- 승률 95%라도 승/패 비율이 1 미만이면 전체 수익 제한
- 이것이 수익률이 낮은 주요 원인

### 1.3 목표
- **승/패 비율**: 0.7767 → 1.5 이상 (승리 시 평균 수익 > 패배 시 평균 손실)
- **구조**: 승리할 때 크게 이기고, 질 때 적게 지는 구조

## 2. 개선 방안

### 2.1 청산 타이밍 최적화 (가장 효과적)

#### 2.1.1 손절/익절 비율 조정

**현재 문제**
- 손절라인이 너무 느슨함 (패배 시 평균 손실 3.19 포인트)
- 익절라인이 너무 빠름 (승리 시 평균 수익 2.48 포인트)

**개선 방안**
- **손절라인 강화**: 3.19 포인트 → 1.5 포인트 (53% 감소)
- **익절라인 완화**: 2.48 포인트 → 4.0 포인트 (61% 증가)
- **예상 승/패 비율**: 4.0 / 1.5 = 2.67

**구현 방법**
```python
# 손절/익절 비율 파라미터 추가
SLIPPAGE_RATIO = 1.5  # 손절라인 (포인트)
PROFIT_RATIO = 4.0    # 익절라인 (포인트)
```

**기대 효과**
- 승/패 비율: 0.7767 → 2.67 (+244%)
- 승률: 95% → 85% (감소 예상)
- 전체 수익: 대폭 증가 예상

#### 2.1.2 ATR 기반 동적 손절/익절

**개선 방안**
- ATR(ATR14)을 기준으로 동적 손절/익절 설정
- 변동성이 높을 때는 더 넓은 범위, 낮을 때는 더 좁은 범위

**구현 방법**
```python
# ATR 기반 동적 손절/익절
atr_multiplier_stop = 1.0  # ATR * 1.0 손절
atr_multiplier_profit = 2.0  # ATR * 2.0 익절
```

**기대 효과**
- 시장 변동성에 맞는 유연한 손절/익절
- 승/패 비율 개선

#### 2.1.3 트레일링 스톱 (Trailing Stop)

**개선 방안**
- 승리 시 이익을 보호하면서 더 큰 수익 추구
- 손실 시 빠른 손절

**구현 방법**
```python
# 트레일링 스톱
trailing_stop_pct = 0.5  # 0.5% 트레일링 스톱
```

**기대 효과**
- 승리 시 평균 수익 증가
- 패배 시 평균 손실 감소

#### 5.1.1 손절/익절 비율 조정
```python
# ml_exit_timing.py 수정
def optimize_exit_timing_with_ratio(df: pd.DataFrame, model: Sequential, 
                                    X: pd.DataFrame, scaler: MinMaxScaler,
                                    stop_loss_pts: float = 1.5,
                                    take_profit_pts: float = 4.0) -> pd.DataFrame:
    """손절/익절 비율 조정 청산 타이밍 최적화"""
    # 기존 청산 타이밍 최적화 로직
    df_optimized = optimize_exit_timing(df, model, X, scaler)
    
    # 손절/익절 비율 조정
    df_optimized = df_optimized[
        (df_optimized['net_pts'] >= -stop_loss_pts) | 
        (df_optimized['net_pts'] >= take_profit_pts)
    ]
    
    return df_optimized
```
